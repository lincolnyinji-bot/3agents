# VC猎头简历筛选系统

## 项目概述
为VC猎头王桢开发的自动化简历筛选系统，专门针对投资经理岗位。

## 筛选标准
- 学历：985/QS100理工科本硕（本科商科排除）
- 经验：1-5年一级股权投资经验
- 薪资：20-35K/月
- 技能：行研、尽调、投成案例等

## 文件结构
- `data/` - 数据文件（学校数据库、测试简历等）
- `scripts/` - Python脚本
- `output/` - 输出文件（推荐列表、报告等）
- `logs/` - 运行日志

## 使用流程
1. 从猎聘网导出简历CSV到`data/input/`
2. 运行筛选脚本：`python scripts/resume_filter.py`
3. 查看输出：`output/recommended_candidates.csv`
4. 手动导入到飞书表格

## 评分规则
总分100分，≥75分推荐
- 学历匹配：40分
- 经验匹配：30分
- 薪资匹配：10分
- 技能匹配：20分
