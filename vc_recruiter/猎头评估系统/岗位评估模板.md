# 🎯 猎头评估系统 - 岗位模板库

## 📋 系统概述
这是基于泛硬科技投资评估经验建立的**可复用评估系统**。每个新岗位只需**选择模板 + 微调参数**即可使用。

---

## 🏗️ 核心架构

### **1. 通用评估框架（所有岗位共用）**
```
输入：候选人简历 → 提取关键信息 → 模板评估 → 输出：匹配结果
```

### **2. 模块化设计**
- **信息提取模块**：处理PDF/图片简历，提取关键信息
- **模板匹配模块**：根据岗位选择评估模板
- **权重调整模块**：微调评估参数
- **结果输出模块**：生成评估报告

### **3. 知识持久化**
- 所有评估标准固化在文件中
- 不会随会话丢失
- 可随时调用和修改

## 🔄 工作流程

### **步骤1：岗位识别**
```
用户输入："我需要评估一个消费投资候选人"
系统识别 → 选择"消费投资"模板
```

### **步骤2：参数微调**
```
用户输入："客户比较苛刻，需要一线机构经验"
系统调整 → 提高"机构经验"权重，设置更严格标准
```

### **步骤3：批量评估**
```
上传简历 → 自动处理 → 模板评估 → 输出结果
```

### **步骤4：结果优化**
```
评估结果 → 人工复核 → 反馈调整 → 模板优化
```

## 📈 效率对比

### **传统方式（每次从零开始）**
```
新岗位 → 重新定义标准 → 测试优化 → 建立评估方法
耗时：2-3小时/岗位
风险：标准不一致，易出错
```

### **本系统方式**
```

耗时：5-10分钟/岗位
优势：标准统一，快速复用
```

## 🔧 使用方法

### **1. 新增岗位模板**
```bash
# 基于现有模板创建新岗位
python create_new_template.py --base=泛硬科技投资 --name=新能源投资
```

### **2. 批量评估简历**
```bash
# 使用指定模板评估简历
python batch_evaluate.py --template=泛硬科技投资 --resumes=简历文件夹
```

### **3. 调整评估参数**
```bash
# 微调模板参数
python adjust_template.py --template=消费投资 --param=商业感知权重 --value=0.6
```
