# 🚀 猎头评估系统 - 使用指南

## 📖 快速开始

### **场景1：评估新岗位候选人**
```bash
# 第1步：选择模板
python select_template.py --岗位=泛硬科技投资

# 第2步：评估简历
python evaluate.py --template=泛硬科技投资 --resume=候选人.pdf

# 第3步：查看结果
cat 评估结果_候选人.md
```

### **场景2：批量评估多个候选人**
```bash
# 一次性评估文件夹内所有简历
python batch_evaluate.py --template=泛硬科技投资 --folder=简历文件夹/
```

### **场景3：创建新岗位模板**
```bash
# 基于现有模板创建
python create_template.py --base=泛硬科技投资 --new=新能源投资
```

## 🔧 实际操作示例

### **示例1：评估消费投资候选人**
```bash
# 如果已有消费投资模板
python evaluate.py --template=消费投资 --resume=消费候选人.pdf

# 如果无模板，快速创建
python create_template.py --base=泛硬科技投资 --new=消费投资
# 然后调整参数（降低技术权重，提高商业感知权重）
```

### **示例2：同时评估多个岗位**
```bash
# 方法1：串行处理
python evaluate.py --template=泛硬科技投资 --resume=A.pdf
python evaluate.py --template=消费投资 --resume=B.pdf

# 方法2：批量处理不同岗位
python multi_evaluate.py \
  --candidates=候选人列表.csv \
  --templates=岗位模板映射.json
```

## 💡 高效工作流

### **第一步：整理需求**
```
客户需求 → 转换为评估标准 → 选择/创建模板
```

### **第二步：收集简历**
```
简历来源 → PDF/图片格式 → 存放指定文件夹
```

### **第三步：批量评估**
```
python batch_evaluate.py --template=XX投资 --folder=简历/
```

### **第四步：结果筛选**
```
评估结果 → 按分数排序 → 生成推荐名单
```

### **第五步：反馈优化**
```
客户反馈 → 调整模板参数 → 优化评估准确性
```

## 🛠️ 故障排除

### **问题1：PDF无法读取**
```bash
# 检查OCR工具
tesseract --version
pdftotext --version

# 重新安装
apt-get install -y tesseract-ocr-chi-sim poppler-utils
```

### **问题2：评估结果不准确**
```bash
# 调整模板参数
python adjust_template.py --template=泛硬科技投资 --param=技术权重 --value=0.5

# 重新评估
python evaluate.py --template=调整后的模板 --resume=候选人.pdf
```
