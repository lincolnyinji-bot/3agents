- 地区：根据需求设置

### 第三步：页面信息采集
**不需要点开详细简历**，直接复制列表页信息：

每行候选人包含：
```
1. 姓名
2. 当前职位（如：投资经理）
3. 当前公司（如：红杉资本）
4. 工作年限（如：3年）
5. 学历（如：硕士）
6. 期望薪资（如有显示）
```

### 第四步：手动填写模板

创建Excel/CSV文件，包含以下列：
```
姓名,当前职位,当前公司,工作年限,学历,期望薪资
```

**示例填写：**
```
张三,投资经理,IDG资本,3年,硕士,25K
李四,分析师,高瓴资本,2年,硕士,22K
王五,投资经理,某VC基金,4年,硕士,30K
```

### 第五步：保存并发送
1. 保存为 `猎聘数据_日期.csv`
2. 发送给我处理
3. 我自动补充专业、经验等详细信息

## 采集数量建议
- **第一次测试**：10-20份（快速验证）
- **日常运行**：50-100份/天
- **目标**：每天筛选出5+有效候选人

## 注意事项
1. **不要频繁刷新**：避免账号风险
2. **分批采集**：每采集20份休息几分钟
3. **优先顺序**：先采集显示信息完整的候选人
4. **标注疑问**：不确定的信息标注"待确认"

## 效率技巧
1. **使用分页**：猎聘网每页15-20人
2. **快捷键**：Ctrl+C复制，Ctrl+V粘贴
3. **批量操作**：可打开多个标签页同时查看
