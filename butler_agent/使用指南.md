# 👔 管家Agent使用指南

## 🎯 概述
管家Agent是自动化猎头系统的**工作流协调中心**，负责：
1. **协调小蜜蜂和裁判Agent**的完整工作流
2. **管理候选人状态**从发现到录用的全流程
3. **定时任务调度**（严格/宽松快筛、日报生成）
4. **报告生成和通知**（每日/每周报告）

## 📁 文件结构
```
butler_agent/
├── butler_agent.py      # 主程序（27945行，完整功能）

```

## 🔧 核心功能

### **1. 工作流协调**
```python
from butler_agent import ButlerAgent, ClientType

# 初始化Agent
agent = ButlerAgent()

# 手动处理JD
jd_text = "松禾资本-人工智能投资经理-深圳"
result = agent.process_jd_manual(jd_text, ClientType.STRICT)

print(f"发现候选人: {result['candidates_discovered']}")
print(f"推荐候选人: {result['candidates_recommended']}")
```

### **2. 定时任务调度**
```python
# 启动定时任务（自动执行）
agent.start()

# 定时任务包括：
# - 09:00：严格快筛（苛刻客户）
# - 20:00：宽松快筛（宽松客户）
# - 22:00：生成日报
```

### **3. 状态管理**
```python
# 获取管道状态
pipeline = agent.get_pipeline_status()
print(f"总候选人: {pipeline['total_candidates']}")
print(f"推荐待审: {pipeline['status_counts']['recommended']}")

# 获取推荐列表
recommendations = agent.get_recommendations(limit=10)
for rec in recommendations:
    print(f"{rec['name']} - 评分: {rec['score']}")
```

### **4. 报告生成**
```python
# 生成日报
report = agent.get_daily_report()
print(f"日报日期: {report['date']}")
print(f"今日发现: {report['summary']['total_discovered']}")
print(f"今日推荐: {report['summary']['recommended']}")

# 推荐候选人详情
for rec in report['recommendations']:
    print(f"- {rec['name']} ({rec['position']})")
```

## 🔄 候选人状态流转

### **完整状态流程**：
```
发现 (discovered) 
  ↓
初筛通过 (screened) 
  ↓
评估完成 (evaluated) 
  ↓
推荐待审 (recommended)  ← 用户审核点
  ↓
用户审核 (reviewed) 
  ↓
联系中 (contacting) 
  ↓
面试安排 (interviewing)
  ↓
录用流程 (hiring)
  ↓
已拒绝 (rejected)
```

## 🐛 故障排除

### **常见问题**：

1. **Agent连接失败**
   ```
   检查：config.yaml中的endpoint配置
   解决：确认小蜜蜂/裁判Agent正在运行
   ```

2. **定时任务不执行**
   ```
   检查：系统时间是否正确
   解决：确认时区设置（Asia/Shanghai）
   ```

3. **状态数据丢失**
   ```
   检查：candidate_status.json文件权限
   解决：启用备份配置，定期备份
   ```

4. **内存占用过高**
   ```
   检查：候选人数据量
   解决：启用cleanup_enabled，定期清理旧数据
   ```

## 🔐 数据安全

### **敏感数据处理**：
1. **候选人信息**：存储在本地JSON文件
2. **API密钥**：不在代码中硬编码，使用配置文件
3. **日志信息**：不记录敏感个人信息

### **备份策略**：
1. **自动备份**：启用backup_enabled
2. **定期清理**：启用cleanup_enabled，清理90天前数据
3. **数据导出**：定期导出重要数据
