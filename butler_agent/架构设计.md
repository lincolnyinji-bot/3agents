# 👔 管家Agent架构设计

## 🎯 设计目标
**工作流协调 + 状态管理 + 定时调度 + 报告生成**

## 🏗️ 系统架构

### **核心模块**：
```
1. 📊 工作流协调器 (WorkflowCoordinator)
   ├── 调用小蜜蜂Agent进行候选人发现
   ├── 调用裁判Agent进行评估
   └── 管理完整流程状态

2. 🗃️ 状态管理器 (StatusManager)
   ├── 候选人状态跟踪（发现→初筛→评估→推荐）
   ├── 工作流状态监控
   └── 异常状态处理

3. ⏰ 任务调度器 (TaskScheduler)
   ├── 严格快筛任务（09:00）
   ├── 宽松快筛任务（20:00）
   └── 报告生成任务（每日/每周）

4. 📈 报告生成器 (ReportGenerator)
   ├── 每日发现报告
   ├── 每周趋势报告
   └── 候选人推荐报告

5. 🚨 异常处理器 (ErrorHandler)
   ├── 网络异常处理
   ├── API调用失败重试
   └── 系统监控告警
```

## 🔄 完整工作流程

### **自动化发现流程**：
```
1. ⏰ 任务调度器触发（09:00严格快筛）
2. 🐝 调用小蜜蜂Agent发现候选人
   ├── JD解析 → 搜索策略
   ├── 执行搜索 → 获取候选人
   ├── 严格初筛 → 过滤候选人
   └── 记忆去重 → 避免重复
3. ⚖️ 调用裁判Agent评估候选人
   ├── 简历深度评估
   ├── 岗位匹配度评分
   └── 生成评估报告
4. 📊 管家Agent汇总结果
   ├── 更新候选人状态
   ├── 生成每日报告
   └── 准备推荐列表
5. 👤 用户审核推荐
   ├── 查看推荐候选人
   ├── 标记后续行动
   └── 反馈校准数据
```

### **候选人状态流转**：
```
发现 (Discovered) 
  ↓
初筛通过 (Screened) 
  ↓
评估完成 (Evaluated) 
  ↓
推荐待审 (Recommended) 
  ↓
用户审核 (Reviewed) 
  ↓
联系中 (Contacting) 
  ↓
面试安排 (Interviewing)
  ↓
录用流程 (Hiring)
```

## 🛠️ 技术实现

### **工作流协调器**：
```python
class WorkflowCoordinator:
    def __init__(self, scout_agent, judge_agent):
        self.scout = scout_agent
        self.judge = judge_agent
        self.status_manager = StatusManager()
    
    def process_jd(self, jd_text, client_type):
        """处理单个JD的完整流程"""
        # 1. 小蜜蜂发现
        candidates = self.scout.process_jd(jd_text, client_type)
        
        # 2. 裁判评估
        evaluations = []
        for candidate in candidates:
            evaluation = self.judge.evaluate(candidate)
            evaluations.append(evaluation)
            
        # 3. 状态更新
        self.status_manager.update_candidates(candidates, evaluations)
        
        return {
            "candidates": candidates,
            "evaluations": evaluations,
            "summary": self._generate_summary(candidates, evaluations)
        }
```

### **任务调度器**：
```python
class TaskScheduler:
    """定时任务调度"""
    
    def __init__(self):
        self.schedule = schedule
        
    def setup_daily_tasks(self):
        """设置每日任务"""
        # 早上9点严格快筛
        schedule.every().day.at("09:00").do(
            self.run_strict_screening
        )
        
        # 晚上8点宽松快筛  
        schedule.every().day.at("20:00").do(
            self.run_loose_screening
        )
        
        # 晚上10点生成日报
        schedule.every().day.at("22:00").do(
            self.generate_daily_report
        )
```
