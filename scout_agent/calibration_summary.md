## 🧠 用户决策逻辑深度分析

基于用户反馈和标注数据，总结用户真实决策逻辑：

### **1. 学历权重体系**：
```
苛刻客户：
第一学历：理工科必须，商科直接拒绝
第二学历：理工科STEM > 商科 > 边缘理工科

宽松客户：
第一学历：可以放宽
第二学历：STEM > 商科 > 边缘理工科
```

### **2. 专业相关性价值**：
```
STEM核心专业（强烈加分）：
- 计算机、软件工程、人工智能、机器学习
- 电子工程、电气工程、通信工程
- 机械工程、航空航天、材料科学

STEM相关专业（适度加分）：
- 物理学、数学、统计学
- 生物医学工程、化学工程

边缘理工科（有限价值）：
- 生物科学（硬科技投资价值有限）
- 土木工程（传统行业应用）
- 环境工程、工业工程

商科专业（苛刻客户直接拒绝）：
- 金融、经济、管理、市场营销
```

### **3. 工作经验价值**：
```
强烈加分（同一机构连续工作）：
- 5年以上：强烈加分（可以弥补学历缺陷）
- 3年以上：显著加分
- 累计4年以上：适度加分

减分项：
- 2年内频繁跳槽：减分
- 非投资相关岗位：价值有限
```

### **4. 特殊情况处理**：
```
缺失第一学历：
- 如果第二学历优秀（985/QS100理工科）
- 且工作经验优秀（3年以上连续工作）
- 可以假设第一学历符合要求
```
